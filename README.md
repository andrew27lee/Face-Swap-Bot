# FaceSwap
Swap the face from one image onto the face(s) of another image or your own face using Python 3 with OpenCV and dlib.

## Deploy Using Docker
```
$ docker build -t <CUSTOM_IMAGE_NAME> .
$ docker run -d -p 5000:5000 <CUSTOM_IMAGE_NAME>
```
